Version 1.0 of D&D DM Dashboard
This program is to be used by a DM to track their party
As well as generating level appropriate encounters

Version 1.0 is a proof of concept, 2.0 will flesh out functionality
As well as the data files


Compilation command:
g++ -std=c++11 -o dndhelp *.cpp
Run:
./dndhelp
